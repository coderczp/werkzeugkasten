Configure Launching GUI based Options

  • Use DbLauncher
    If you want to use DbLauncher check this.
  • Launch Debugger
    If enabled, DbLauncher launches Database by debugger.
  • Open Console in eclipse window.
    If enabled, DbLauncher opens Console by browser in eclipse window.
  • Add JDBC Driver Jar To Build Path.
    • If enabled, add JDBC Driver variable classpath entry to project.
    • If disabled, remove JDBC Driver variable classpath entry from project.
  • Db Type
    Select the RDB.
  • Db PortNo
    Specify the port of TCP Server.
  • Console PortNo
    Specify the port of Web Server for Console.
  • User
    Specify the string of user id for Database.
  • Password
    Specify the string of password for Database.
  • Base Dir
    Specify the base directory for database files
