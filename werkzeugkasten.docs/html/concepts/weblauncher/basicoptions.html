Configure Launching GUI based Options

  • Use WebLauncher
    If you want to use WebLauncher check this.
  • Add Servlet API Jar To Build Path.
    • If enabled, add Servlet API variable classpath entry to project.
    • If disabled, remove Servlet API variable classpath entry from project.
  • Web Server
    Select the Servlet Container.
  • Context Name
    Specify the context name of Web Application.
  • BaseDir
    Specify the directory of Web Application root.
  • Web PortNo
    Specify the port of Web Server.
  • Config File
    Specify the configuration file of Web Server.
  • Check Server When Open Url
    If enabled, When View on Server, WebLauncher check Web Server stopped then WebLauncher launches Web Server.
  • Launch Debugger
    If enabled, WebLauncher launches Web Server by debugger.
  • Open Browser in eclipse window.
    If enabled, WebLauncher opens contains by browser in eclipse window.
