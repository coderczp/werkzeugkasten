DbLauncherの使い方

  1. Package Explorerでプロジェクトを選択する。
  2. プロジェクトのコンテキストメニューからプロパティ を選択する。
  3. プロジェクトのプロパティページの中から DbLauncher を選択する。
  4. H2 データベースを使うをチェックする。
  5. OKを押下する。
  6. H2 データベース を起動するには３種類の方法がある。
    • プロジェクトのコンテキストメニューから DbLauncher > H2 データベースを起動する を選択する。
    • メニューバーから DbLauncher > H2 データベースを起動する を選択する。
    • ツールバーの H2 データベースを起動するボタンをクリックする。
  7. Dblauncher が H2 データベースを起動に成功すると, プロジェクトのアイコンとラベルが以下の様に装飾される。
  8. H2 データベース を停止するには３種類の方法がある。
    • プロジェクトのコンテキストメニューから DbLauncher > H2 データベースを停止する を選択する。
    • メニューバーから DbLauncher > H2 データベースを停止する を選択する。
    • ツールバーの H2 データベースを停止するボタンをクリックする。
