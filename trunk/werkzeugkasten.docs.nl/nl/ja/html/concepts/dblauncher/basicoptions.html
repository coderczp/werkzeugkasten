Configure H2 Launching GUI based Options

  • H2 データベースを使う
    DbLauncherを使用する際には必ずチェックする事。
  • デバッガで起動する
    チェックすると、DbLauncher が H2 データベースを起動する際にデバッガを使用する。
  • H2 コンソールをeclipseウインドウ内で開く
    チェックすると、DbLauncherはeclipseのウィンドウ内で H2 コンソール開く。
  • H2 のJDBCドライバをビルドパスに追加する
    • チェックすると、H2のライブラリをプロジェクトのクラスパスエントリに追加する。
    • チェックを外すと、H2のライブラリをプロジェクトのクラスパスエントリから削除する。
  • DBサーバポート番号
    TCP サーバのポート番号を設定する。
  • H2 コンソールポート番号
    主にH2 コンソールで使用するWebサーバのポート番号を設定する。
  • ユーザＩＤ
    H2 データベースで使用するユーザＩＤを設定する。
  • パスワード
    H2 データベースで使用するパスワードを設定する。
  • データ格納ディレクトリ
    データベースのファイルを格納する為のディレクトリを設定する。
