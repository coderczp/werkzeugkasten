Configure H2 Launching GUI based Options

  • Use H2 Database
    If you want to use DbLauncher check this.
  • Launch Debugger
    If enabled, DbLauncher launches H2 Database by debugger.
  • Open H2 Console in eclipse window.
    If enabled, DbLauncher opens H2 Console by browser in eclipse window.
  • Add H2 JDBC Driver Jar To Build Path.
    • If enabled, add H2 variable classpath entry to project.
    • If disabled, remove H2 variable classpath entry from project.
  • Db PortNo
    Specify the port of TCP Server.
  • H2 Console PortNo
    Specify the port of Web Server for H2 Console.
  • User
    Specify the string of user id for H2 Database.
  • Password
    Specify the string of password for H2 Database.
  • Base Dir
    Specify the base directory for database files
