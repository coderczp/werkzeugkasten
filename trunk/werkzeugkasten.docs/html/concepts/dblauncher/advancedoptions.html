Configure Launching Advanced Options

  1. At First, Launching Database by DbLauncher
  2. DbLauncher define launch configuration for H2 Database.
  3. Select Run > Run... or Run > Debug... from the workbench Run menu to open the list of launch configurations.
  4. Select an existing configuration defined by DbLauncher
  5. On the Arguments tab for the configuration, you can specify the parameters
